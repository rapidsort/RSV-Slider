RSV Slider is a jQuery slider.

== Description ==

RSV Slider is  based on <a href="http://owlgraphic.com/owlcarousel/">OWL Carousel</a>. It uses a media images for each slide to create slider groups. you can fill your pages with sliders.

== Installation ==

1. Upload the <strong>rs-slider</strong> folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode <strong>[rs_slider id="Your_ID" images="image1(URL),image2(URL),image3(URL)"]</strong> in the content area of a page or post where you want the image slider to appear.
4. Change <strong>id</strong> for every shortcode used.  
5. <strong>id</strong> can be any text with out spaces.
6. Add multiple images (URL's) in <strong>image</strong> Property by separating comma.


== Frequently Asked Questions ==

= How do I insert the RSV Slider? =
You can insert the RSV Slider by pasting the shortcode `[rs_slider id="Your_ID" images="image1,image2,image3"]` into the content area of a page or post.  **Be sure to use the HTML editor when inserting shortcodes!** To customize your slider, there are optional attributes that can be used with the shortcode.
